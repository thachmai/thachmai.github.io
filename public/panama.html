Why Data Viz?

We're not naturally equipped to deal with numbers
Data vizualization allows us to "see" the data using existing facilities: shape, area, volume, color, orientation, etc
Data vizualization is a communication technique, not an artistic expressions

Data Viz today

D3.js dominates the landscape (for good reasons ?)
Mosly charts - Still rare to see interactions and transitions
Effective data vizualization requires an aggregation of various techniques: web dev, 2D/3D graphics, data analysis
Opportunities?

Wanna try?

I recommend starting with D3.js
Play around with the [examples](https://bl.ocks.org/mbostock)
D3.js is mostly functional programming, so familiarity with the paradigm helps
Understand the data join ENTER UPDATE EXIT contract!

What else?

Data scrapping helps. I use Clojure with Enlive library
Understanding statistics allows you to tell a story with data
Avoid misleading your readers
